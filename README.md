# Cài đặt

## Clone project

```bash
git clone https://github.com/hanvaz/todolist.git
```

## Cài đặt Backend

```bash
cd server
npm install
```

## Chạy Backend

```bash
npm start
```

# Các package chính

Frontend:

* React
* Axios

Backend:

* Express
* CORS
* UUID

Nếu sử dụng MySQL thì cài thêm:

npm install mysql2
